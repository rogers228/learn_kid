# 檔案概念
有分1.檔案，2.資料夾


檔案file 是由 name.副檔名   所組成
1. 檔案名稱 name
2. .點
3. 副檔名, 檔案類型  extension



## 隱藏副檔名
有的檔案總管會預設隱藏副檔名

## 檔案儲存的時候要指定附檔名
有的沒指定是因為 有些程式會幫你設定

## 檔案的位置
檔案是放在電腦裡的硬碟，從檔案總管app(程式)去開來看 去找
檔案的位置就是資料夾

## 資料夾
資料夾裡面可以放檔案，也可以放資料夾
資料夾就是整理分類檔案用的，以後才好找
資料夾他是一個範圍，不是這類的範圍不要放進來
資料夾一開始就要設定好她的範圍


## 檔案在哪裡
完整位置，完整路徑 就是a/b/c/file.py
